# Losam
# 🛍️ Site de Shopping en Ligne

Ce projet est un site web front-end développé avec **HTML, CSS et JavaScript**.

Il s'agit d’un site de shopping en ligne où sont présentés des produits pour femmes comme le maquillage, les sacs et d'autres accessoires.

Le site contient différentes catégories et une interface simple et responsive.  
Aucune base de données ni backend n’est utilisé pour le moment.

## 🛠️ Technologies
- HTML
- CSS
- JavaScript
